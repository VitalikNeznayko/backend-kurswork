# STUDENT_DOCS

## ВСТУП

### Актуальність теми
У сучасному цифровому середовищі онлайн-форуми залишаються важливим інструментом для комунікації, обміну знаннями та створення спільнот. Веб-форуми дозволяють користувачам обговорювати теми, ділитися досвідом та отримувати допомогу.

---

### Мета роботи
Розробка веб-додатку типу форуму з використанням архітектури MVC для організації обговорень між користувачами.

---

### Завдання роботи
- Реалізувати систему реєстрації та авторизації користувачів  
- Створити структуру форуму (категорії, теми, пости)  
- Реалізувати можливість додавання та редагування постів  
- Додати систему коментарів  
- Реалізувати базову систему адміністрування  

---

### Об’єкт дослідження
Веб-додатки для організації онлайн-комунікації.

---

### Предмет дослідження
Методи розробки веб-додатків із використанням архітектури MVC та взаємодії з базою даних.

---

### Практичне значення
Результатом роботи є функціональний форум, який може бути використаний як основа для створення повноцінної платформи спілкування.

---

## РОЗДІЛ 1. ТЕХНОЛОГІЧНИЙ СТЕК

Для розробки веб-додатку було використано наступні технології:

### Backend
Основною мовою програмування є PHP.  
Реалізовано архітектуру MVC, що дозволяє розділити логіку додатку на окремі компоненти:
- Моделі відповідають за роботу з даними  
- Контролери обробляють запити користувача  
- Представлення відповідають за відображення  

---

### База даних
Для зберігання даних використовується MySQL.  
База даних містить таблиці:
- Users (користувачі)
- Categories (категорії)
- Topics (теми)
- Posts (пости)
- Comments (коментарі)

Взаємодія з базою реалізована через PDO з використанням підготовлених запитів.

---

### Frontend
Використовуються:
- HTML для структури сторінок  
- CSS для стилізації  
- JavaScript для динамічної взаємодії  

Також застосовується AJAX для асинхронного обміну даними без перезавантаження сторінки.

---

### Серверне середовище
Додаток працює на локальному сервері Apache (XAMPP).  
Використовується файл index.php як точка входу (Front Controller).

---

## ВИСНОВКИ

У ході виконання роботи було розроблено веб-форум із використанням архітектури MVC.  
Система дозволяє користувачам взаємодіяти через теми, пости та коментарі.

Було реалізовано:
- систему авторизації  
- структуру форуму  
- CRUD-операції для контенту  
- базову безпеку  

Отримані знання можуть бути використані при створенні більш складних веб-додатків.
